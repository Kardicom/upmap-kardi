# Руководство по кастомным компонентам Tina CMS

После перезапуска dev сервера в редакторе Tina CMS появятся новые кнопки для вставки блоков.

## Доступные компоненты:

### 1. **ChecklistBlock** — Чёрный блок с галочками
Используется для чеклистов, списков требований.

**Как вставить:**
- В редакторе нажми на `+` (добавить компонент)
- Выбери "Чёрный блок с галочками"
- Внутри добавь список с галочками

**Пример в MDX:**
```mdx
<ChecklistBlock>

✅ Правильное название (БЕЗ ключевых слов в имени компании)
✅ Все категории заполнены (основная + 4 дополнительных)
✅ Описание 500+ символов с ключами

</ChecklistBlock>
```

---

### 2. **InfoBox** — Информационный блок (синий)
Для важной информации, заметок.

```mdx
<InfoBox title="Важно знать">

Яндекс удалит отзыв ТОЛЬКО в следующих случаях...

</InfoBox>
```

---

### 3. **WarningBox** — Предупреждение (жёлтый)
Для предостережений, важных моментов.

```mdx
<WarningBox title="Внимание">

Не указывайте конкретные размеры шин в описании!

</WarningBox>
```

---

### 4. **SuccessBox** — Успех (зелёный)
Для положительных результатов, советов.

```mdx
<SuccessBox title="Отлично">

Вы успешно настроили карточку!

</SuccessBox>
```

---

### 5. **TipBox** — Совет (фиолетовый)
Для полезных советов, лайфхаков.

```mdx
<TipBox>

Используйте автоматический сбор отзывов через QR-код

</TipBox>
```

---

### 6. **QuoteBox** — Цитата с автором
Для цитат клиентов, экспертов.

```mdx
<QuoteBox author="Иван Петров, владелец СТО">

После оптимизации карточки количество звонков выросло в 3 раза

</QuoteBox>
```

---

### 7. **HighlightText** — Выделенный текст (жёлтый маркер)
Для акцента на важных словах прямо в тексте.

```mdx
Это <HighlightText>очень важная</HighlightText> информация!
```

---

### 8. **ButtonLink** — Кнопка-ссылка
Для призывов к действию (CTA).

```mdx
<ButtonLink href="https://upmap.ru/kontakty">

Заказать аудит карточки

</ButtonLink>
```

---

## Как использовать в Tina CMS:

1. **Откройте редактор статьи** в Tina CMS
2. **Наведите курсор** на место, куда хотите вставить блок
3. **Нажмите кнопку `+`** (добавить компонент)
4. **Выберите нужный компонент** из списка
5. **Заполните поля** (заголовок, текст, ссылка)
6. **Сохраните**

---

## В обычном MDX-файле:

Можно просто написать компонент вручную:

```mdx
---
title: "Моя статья"
date: "2025-11-02"
---

## Введение

Обычный текст статьи.

<TipBox>
Полезный совет для читателей!
</TipBox>

<ChecklistBlock>

✅ Пункт 1
✅ Пункт 2
✅ Пункт 3

</ChecklistBlock>

<ButtonLink href="https://example.com">
Перейти на сайт
</ButtonLink>
```

---

## Советы:

- Используйте **ChecklistBlock** для списков требований
- **InfoBox / WarningBox / SuccessBox** — для важных блоков информации
- **TipBox** — для лайфхаков и советов
- **QuoteBox** — для отзывов клиентов
- **ButtonLink** — для призывов к действию (CTA)

Все компоненты адаптивные и хорошо выглядят на мобильных устройствах!
